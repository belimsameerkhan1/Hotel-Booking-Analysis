# 🏨 Hotel Booking Data Analysis Dashboard

# 📊 Project Overview
This project transforms a raw dataset of over 119,000 hotel bookings into an interactive visual dashboard. 

# 🛠️ Tools Used
Microsoft Excel:** Data Cleaning, Pivot Tables, Slicers.
 Data Visualization:** Custom UI/UX design (Lavender-Cream Theme).

# 📈 Key Insights
High Churn:** Identified a 37% overall cancellation rate.
Segment Analysis:** Couples are the primary booking demographic.
* **Seasonality:** September peak identified for strategic planning.
