# Hotel-Booking-Analysis
📊 From 119k+ Rows to Actionable Insights  I've just wrapped up a deep dive into Hotel Booking trends (2015-2017).  The Challenge: Processing a massive raw dataset with missing values. The Solution: Cleaned the data and built a dynamic Excel dashboard using Pivot Tables and Slicers. Key Insight: Identified a 37% cancellation rate, with specific peaks in September and higher churn in City Hotels.
